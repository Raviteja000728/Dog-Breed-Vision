## Dog Breed Detector 🐾

An AI-powered web application that detects dog breeds from uploaded images using TensorFlow and Streamlit. The app uses deep learning with transfer learning models like MobileNetV2, EfficientNetB0, ResNet50, and VGG16 for accurate breed prediction.

## Features
* Upload dog images for instant breed detection
* Beautiful Streamlit UI with animations
* Multiple pretrained model backbones
* Fast image preprocessing and prediction
* Responsive modern design
